# KUROKO PLUS LP用情報まとめ

## 製品名
- **KUROKO PLUS**（旧名: ステルスAI）

## キャッチコピー
- 「まわりの会話をそっと聞いて、あなただけに情報を」
- 「会話をもっと楽しく、もっと便利にする新しいAIおともだち」
- 「会話に寄り添う、あなただけのAI相棒」

## コンセプト
- 雑談から会議まで、会話の空気を壊さずそっと寄り添う**会話支援AI**
- スマホ・タブレット・PCなど、どんな場面でも使いやすいデザイン
- 操作不要・邪魔しない・会話を助ける**ステルス知能**

## 7つの主要機能
1. **都度要約機能** - 会話の流れをつかみやすく自動で要約
2. **瞬間用語解説機能** - 難しい用語を一言で分かりやすく説明
3. **ネット検索連携機能** - 会話内容に関連する情報を自動検索して表示
4. **ベスト質問＆回答提示機能** - 適切な質問や回答例をリアルタイムで提案
5. **声色分析＆人物識別機能** - 話者を自動識別し、性格傾向まで分析
6. **主観的レポート生成機能** - 会話をあなた目線でマイノート感覚にまとめる
7. **矛盾検知＆公平記録機能** (NEW) - 発言の矛盾や信頼度を可視化し、「言った言わない問題」を解決

## 利用シーン
- 家族の会話をもっと楽しく
- 友達とのカフェタイム
- 学校での勉強サポート
- オンライン会議もラクラク
- お客様対応もスマートに

## ターゲットユーザー
- 会話が苦手な若者・外国人
- トラブル回避したいビジネスパーソン
- 記録を残したい専門職の方々
- 家族や友達との会話を大切にしたい方

## 将来展望
- スマートイヤホンやARグラスとの連携で装着型AIアシスタントへ進化
- 翻訳・音声合成と連動し、国際会話や聴覚支援にも対応
- 匿名モード・プライバシー保護強化で公共空間でも安心して使用可能

## 技術スタック（仕様書より）
- フロントエンド: React 18.2.0 + TypeScript + Vite
- バックエンド: Express
- 音声認識: AssemblyAI, OpenAI Whisper, Web Speech API
- AI解析: Google Gemini API
- デプロイ: Render.com
